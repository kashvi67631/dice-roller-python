# 🎲 Dice Roller – Python

A simple Python script to simulate dice rolling using the `random` module.

## 💡 Features

- Generates random number from 1 to 6
- Loops until user decides to stop
- Clean and beginner-friendly code

## 🛠️ Tech Stack

- Python 3.x
- random module

## 🚀 How to Run

```bash
python dice_roll.py
